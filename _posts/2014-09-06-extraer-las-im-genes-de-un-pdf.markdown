Comando para extraer las imágenes de un archivo pdf usando [pdftk](https://www.pdflabs.com/tools/pdftk-the-pdf-toolkit/).

{% highlight bash %}
$ pdfimage -j archivo.pdf nombre-base-de-las-imagenes
{% endhighlight %}
